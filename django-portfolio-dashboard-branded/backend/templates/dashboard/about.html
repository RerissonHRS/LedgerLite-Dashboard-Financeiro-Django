Objetivo

Demonstrar habilidades práticas em Django com foco em qualidade de código, organização e uma UI amigável a recrutadores.

O que este projeto demonstra
  • Autenticação (Login/Logout), rotas protegidas e permissões básicas
  • Modelagem de dados com Django ORM
  • CRUD completo (criar, listar, editar, excluir) de Vendas
  • Dashboard com Chart.js + KPIs
  • Exportação de dados em CSV e Excel
  • Templates com Bootstrap 5, componentes reutilizáveis e responsividade
  • Testes básicos com TestCase

Contato

  • Stack: Django, Bootstrap 5, Chart.js, SQLite
Como rodar
  1. Crie e ative um venv
  2. Instale dependências com pip install -r requirements.txt
  3. Aplicar migrações e rodar o servidor
